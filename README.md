# ip-geocoder
Find geocoding information from IP address
Powered by [ipapi.co](https://ipapi.co)

## Install

```bash
$ npm install --global ip-geocoder
```


## Usage

```bash

  Usage
    $ ip-geocoder 8.8.8.8
    
      ip-geocoder
      Find geocoding information from IP address
      Powered by ipapi.co
      
      ip:           8.8.8.8
      .......

```
